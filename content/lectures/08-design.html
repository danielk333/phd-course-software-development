Example: Level 2 Container - How do I view the results?

  • Option 1: A server that streams the resulting video
    • No client requirements (except video receive)
    • Server side calculations
    • No user control
  • Option 2: A viewer that reads the data locally
    • Client must compile/run viewer
    • Client must download all data and have sufficient hardware
    • Total user control
