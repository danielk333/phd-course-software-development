Example: Level 2 Container - How do I view the results?

  • Option 1: A server that streams the resulting video
    • No client requirements (except video receive)
    • Server side calculations
    • No user control
  • Option 2: A viewer that reads the data locally
    • Client must compile/run viewer
    • Client must download all data and have sufficient hardware
    • Total user control
  • Which one is better?

Level 4 Code example - How do I represent the data?

                    
                        class Position3D:
                            def __init__(self, x, y, z):
                                self.x = x
                                self.y = y
                                self.z = z


                        class Velocity3D:
                            def __init__(self, vx, vy, vz):
                                self.vx = vx
                                self.vy = vy
                                self.vz = vz


                        class State:
                            def __init__(self, pos: Position3D, vel: Velocity3D):
                                self.position = pos
                                self.velocity = vel


                        class Simulation:
                            def __init__(self, file):
                                self.objects = {}
                                for identifier, state in self.load(file):
                                    self.objects[identifier] = State(
                                        Position3D(state[0], state[1], state[2]),
                                        Velocity3D(state[3], state[4], state[5]),
                                    )
